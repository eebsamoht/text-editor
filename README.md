### Text Editor
A simple plain text editor served as a static HTML5 page. I wrote this for learning purposes, as I'm not particularly well-versed in web design and browser APIs.

### Documentation
The editor uses a straightforward <textarea> element for the page. The only "aftermarket" editing tool included is the ability to create formatted lists. To do this, the user must start a new line with a space and one of the two supported list types: numbers and bullets. When the user presses enter with at least a space after the bullet, the newline will continue the same pattern.

Bullet List Example:
 - List item A
 - List item B

Number List Example:
 1. List item A
 2. List item B
